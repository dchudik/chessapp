## Chesslib/

## Chessapi/
REST API интерфейс для вычислений угрозы королю от ладьи и слона.
API-сервер использует библиотеку **Chesslib/**
Документация в облаке: https://pkg.go.dev/github.com/dchudik/chessapp/chessapi
